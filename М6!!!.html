🚀 Шпаргалка: Модуль 6

Laravel — валидация данных от эмулятора

КАК ЭТО РАБОТАЕТ

На экзамене дают программу-эмулятор TransferSimulator. Она выдаёт данные (ФИО, ИНН и т.д.) через API.

Твоя задача:

1. Кнопка "Получить данные" → берёт данные с эмулятора → показывает на экране

2. Кнопка "Отправить результат теста" → проверяет данные → пишет корректно или нет

💡 Ссылку эмулятора дадут на экзамене. Вставляешь её в контроллер и после слеша пишешь что нужно: fullName, inn, snils и т.д.
РЕГУЛЯРНЫЕ ВЫРАЖЕНИЯ — ЧТО КУДА ВСТАВЛЯТЬ

Регулярное выражение — это шаблон проверки текста. На экзамене скажут что проверять — берёшь нужную строку из таблицы и вставляешь в контроллер.

Что проверяем Регулярное выражение Пример правильного
ФИО /^[А-Яа-яЁё]+ [А-Яа-яЁё]+ [А-Яа-яЁё]+$/u Иванов Иван Иванович
СНИЛС /^[0-9]{3}-[0-9]{3}-[0-9]{3} [0-9]{2}$/ 123-456-789 01
Телефон /^\+7 [0-9]{3} [0-9]{3}-[0-9]{2}-[0-9]{2}$/ +7 999 123-45-67
ИНН /^[0-9]+$/ 1234567890
Паспорт /^[0-9]{2} [0-9]{2} [0-9]{6}$/ 12 34 567890
Email через Laravel: 'email' => 'required|email' ivan@mail.ru
ШАГ 1: Создать проект

Открой терминал и выполни:

composer create-project laravel/laravel --prefer-dist demo6
cd demo6
php artisan make:controller TestCaseController
💡 --prefer-dist скачивает быстрее
ШАГ 2: Настроить маршруты — routes/web.php

Открой routes/web.php и замени всё содержимое на:

<?php
use App\Http\Controllers\TestCaseController;
use Illuminate\Support\Facades\Route;

Route::get('/', [TestCaseController::class, 'show'])->name('test-case');
Route::get('/get', [TestCaseController::class, 'getData'])->name('test-case.get');
Route::post('/check', [TestCaseController::class, 'checkData'])->name('test-case.check');
ШАГ 3: Написать контроллер — app/Http/Controllers/TestCaseController.php

Открой app/Http/Controllers/TestCaseController.php и замени всё на:

💡 Вместо http://localhost:8080/api/ вставь ссылку которую дадут на экзамене. После слеша пишешь что нужно: fullName, inn, snils, phone, passport
<?php
namespace App\Http\Controllers;

use Illuminate\Http\Request;

class TestCaseController extends Controller
{
    // Показать страницу
    public function show(Request $request)
    {
        return view('index');
    }

    // Получить данные с эмулятора
    public function getData()
    {
        // ВСТАВЬ СЮДА ССЫЛКУ С ЭКЗАМЕНА + нужное поле после слеша
        $data = file_get_contents('http://localhost:8080/api/fullName');

        if ($data) {
            $data = json_decode($data)->value;
        }

        return redirect()->route('test-case')->with('value', $data);
    }

    // Проверить данные
    public function checkData(Request $request)
    {
        $value = $request->input('value');

        // ВЫБЕРИ НУЖНУЮ РЕГУЛЯРКУ ИЗ ТАБЛИЦЫ ВЫШЕ И ВСТАВЬ СЮДА:
        if (preg_match('/^[А-Яа-яЁё]+ [А-Яа-яЁё]+ [А-Яа-яЁё]+$/u', $value)) {
            return redirect()->route('test-case')
                ->with('value', $value)
                ->with('message', 'ФИО корректно');
        } else {
            return redirect()->route('test-case')
                ->with('value', $value)
                ->with('message', 'ФИО содержит некорректные символы');
        }
    }
}
⚠️ Если нужно проверять СНИЛС — меняешь только регулярку и текст сообщения! Всё остальное остаётся.

Примеры как менять под разные данные:

// Для СНИЛС:
if (preg_match('/^[0-9]{3}-[0-9]{3}-[0-9]{3} [0-9]{2}$/', $value)) {
    ->with('message', 'СНИЛС корректен');
} else {
    ->with('message', 'СНИЛС некорректен');
}

// Для телефона:
if (preg_match('/^\+7 [0-9]{3} [0-9]{3}-[0-9]{2}-[0-9]{2}$/', $value)) {
    ->with('message', 'Телефон корректен');
} else {
    ->with('message', 'Телефон некорректен');
}

// Для паспорта:
if (preg_match('/^[0-9]{2} [0-9]{2} [0-9]{6}$/', $value)) {
    ->with('message', 'Паспорт корректен');
} else {
    ->with('message', 'Паспорт некорректен');
}

Если нужно получить несколько полей сразу:

public function getData()
{
    $fullName = json_decode(file_get_contents('http://localhost:8080/api/fullName'))->value;
    $inn = json_decode(file_get_contents('http://localhost:8080/api/inn'))->value;

    return redirect()->route('test-case')
        ->with('fullName', $fullName)
        ->with('inn', $inn);
}
ШАГ 4: Создать страницу — resources/views/index.blade.php

Удали файл resources/views/welcome.blade.php

Создай новый файл resources/views/index.blade.php и вставь:

<!DOCTYPE html>
<html lang="ru">
<head>

        .box { border: 1px solid #ccc; padding: 20px; border-radius: 8px; margin-bottom: 20px; }
        button { padding: 8px 20px; background: #4ecca3; border: none; border-radius: 4px; cursor: pointer; font-size: 14px; }
        button:hover { background: #38b2ac; }
        .value { margin-left: 10px; font-weight: bold; color: #1a1a2e; }
        .message { margin-left: 10px; color: green; font-weight: bold; }
        .error { color: red; }
        h2 { color: #1a1a2e; }
    </style>
</head>
<body>
    <h2>Валидация данных</h2>

    <!-- Кнопка 1: Получить данные -->
    <div class="box">
        <form action="{{ route('test-case.get') }}" method="GET">
            <button type="submit">Получить данные</button>
            <span class="value">{{ session('value') }}</span>
        </form>
    </div>

    <!-- Кнопка 2: Отправить результат теста -->
    <div class="box">
        <form method="POST" action="{{ route('test-case.check') }}">
            @csrf
            <input type="hidden" name="value" value="{{ session('value') }}">
            <button type="submit">Отправить результат теста</button>
            <span class="message">{{ session('message') }}</span>
        </form>
    </div>

</body>
</html>
💡 Если нужно вывести несколько полей — добавь в blade дополнительные span:
<span>{{ session('fullName') }}</span>
<span>{{ session('inn') }}</span>
ШАГ 5: Запустить проект
php artisan serve

Открой браузер: http://localhost:8000

✅ Нажми "Получить данные" — должно появиться ФИО с эмулятора. Нажми "Отправить результат теста" — появится сообщение корректно или нет.
Порядок на экзамене: создать проект → routes/web.php → контроллер (вставить ссылку + регулярку) → index.blade.php → serve → проверить
